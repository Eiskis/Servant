In Servant, you can create new web sites from static articles. Just add any html, md or txt files to a new folder under sites/ and it works!

Servant comes in a zip. Download the latest production release to get started:

Download servant.zip

Requires PHP 5.2+

To install, unzip the download on your server. See installation guide for more detailed instructions.

Interested in development? Download a development snapshot to get the newest features and all sample themes, templates and content.
