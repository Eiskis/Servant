
# Installing Servant

This is a detailed walkthrough. For a short version, consult **readme.txt**



## 1. Access a server

You need *PHP 5.2+* and *Apache*. Apache needs to have *rewrite_module* (sometimes called *mod_rewrite*).



## 2. Download Servant

Download Servant. There are two releases:

- **Production release**
	- One theme (*Default*)
	- One template (*Default*)
	- One sample site (just to say hello)
- **Development snapshot**
	- All sample themes
	- All sample templates
	- All sample sites
	- Full documentation
