Servant

A practical, approachable micro web framework. Designed for us humans who frequently whip up new web sites.

PHP + Apache. Easy setup. Open-source.

Read more Download alpha

When you just need a web site

Servant turns your text, Markdown, HTML etc. files into a sitemap, styled with the assets you throw at it. It runs without elaborate configuration, deployment routines or database.

You choose whether to develop your own layout and look or use existing themes and templates for immediate output. Servant scales to your needs – up to a point.

Servant helps you create web sites – it's not a web app platform. Use something else for things like authentication and heavy database operations. You can set up your frontend with Servant even if you use e.g. Play Framework for business logic and APIs.

PHP 5.3+ and Apache with rewrite module required. Drop-in installation, no database.

Features

  • Human-readable URLs
  • Automated asset pipeline
  • Automatic caching
  • Extendable backend actions
  • HTML meta tag generation

Language support

  • CSS, LESS + SCSS styles
  • Plain text, Markdown, Textile, HTML pages
  • PHP, Twig, HAML, Jade scripting

Usable for everyone in our team. Producers, designers and developers can all contribute when we use Servant.

We publish our Markdown documentation with Servant. Docs stay clean for us, users get a usable, beautiful site.

A great lightweight alternative to WordPress or Joomla. It gives us the things we need with no bloat.

I use Servant whenever I don’t need/want a database. And when I do, I still build the frontend with Servant.
